Listas ordenadas

Podemos criar uma lista ordenada utilizando a tag OL e logo após a tag LI para cada item

  1. Item 1
  2. Item 2
  3. Item 3

A tag LI se tornou opcional usar seu fechamento de tag "</li>"

Tem alguns tipos de lista que podemos usar no ol
São eles: 1, A, a, i, I

Um exemplo usando o tipo A

  1. Item 1
  2. Item 2
  3. Item 3

Outro detalhe, posso iniciar de onde eu quero, exemplo: inicar no nº 5, é só colocar start='5' a frente do style

Listas não ordenadas

Para listas não ordenadas usamos o UL

  • Item 1
  • Item 2
  • Item 3

    • Temos 3 opções de tipos de lista não ordenadas: square, circle e disc

Usando links externos

Você pode acessar o meu repositório público no meu git hub

Para abrir o meu repositório em outra página em branco, podemos usar depois da tag a o target="_blank" que indica uma nova pagina em branco;
E logo após o rel="external" para indicar que é um link externo,ou seja fora do meu site.

É recomendado usar links externos para algum site que n seja seu. E pra outras abas do seu site,ou algo parecido, não precisa-se usar link externos que abram outra aba do seu site.
